# Metrics That Matter - Fitness Site

A data-driven, evidence-based fitness education website focusing on the five health metrics that truly impact longevity and quality of life.

## Project Structure

```
Fitness/
├── index.html              # Homepage
├── strength.html           # Strength metric page (fully populated example)
├── vo2max.html            # VO2 Max metric page (template with placeholders)
├── ldl.html               # LDL cholesterol page (template)
├── blood-pressure.html    # Blood pressure page (template)
├── body-fat.html          # Body fat percentage page (template)
├── myths.html             # Debunking fitness myths
├── tracking.html          # Tracking tools and methods
├── about.html             # About the site and philosophy
├── css/
│   ├── main.css          # Main stylesheet with clean, data-focused design
│   └── charts.css        # Chart-specific styles
├── js/
│   ├── navigation.js     # Navigation functionality and smooth scrolling
│   └── charts.js         # Chart.js visualizations with sample data
├── images/               # Directory for images and infographics (empty)
└── data/                 # Directory for research data/JSON (empty)
```

## The Five Key Metrics

1. **Strength** - Muscular strength as a predictor of longevity
2. **VO2 Max** - Cardiovascular fitness and mortality risk
3. **LDL Cholesterol** - Primary driver of cardiovascular disease
4. **Blood Pressure** - Impact on heart, brain, and kidney health
5. **Body Fat Percentage** - Metabolic health beyond body weight

## Viewing the Site

To view the site locally:

1. Open `index.html` in a web browser
2. Navigate using the top navigation bar
3. All pages are linked and functional

No build process or server required - it's a static site that runs directly in the browser.

## Design Philosophy

- **Evidence over anecdote** - All claims backed by research
- **Clarity over complexity** - Simple, accessible information
- **Data-driven** - Visualizations to support key points
- **Mobile-friendly** - Responsive design for all devices
- **No ads, no products** - Pure educational content
