# Graphics & Infographics Guide

This document outlines all the graphics needed for the Metrics That Matter site, with specifications and suggestions for creation.

## Tools for Creating Graphics

### Recommended Options:
1. **Canva** (easiest, templates available) - Free tier works well
2. **Figma** (professional, collaborative) - Free for individuals
3. **Adobe Illustrator** (professional, if available)
4. **Excalidraw** (simple diagrams and flowcharts) - Free
5. **Chart.js** (already implemented for data visualizations)

### For Icons:
- **Heroicons** (free, modern) - https://heroicons.com
- **Font Awesome** (extensive library) - https://fontawesome.com
- **Lucide Icons** (clean, simple) - https://lucide.dev

---

## Infographic Specifications

### Standard Dimensions:
- **Hero images**: 1200x600px
- **Metric cards**: 400x300px
- **Infographics**: 800x1000px (portrait) or 1200x600px (landscape)
- **Icons**: 64x64px or SVG
- **Charts**: Responsive (handled by Chart.js)

### Color Palette (from site design):
- Primary Blue: #2c5282
- Primary Dark: #1a365d
- Primary Light: #4a90e2
- Success Green: #48bb78
- Warning Orange: #ed8936
- Error Red: #ed6464
- Background: #f7fafc
- Text Dark: #1a202c

## Tips for Creating Graphics

### Design Principles:
- **Consistency**: Use the site color palette
- **Clarity**: Simple, easy to understand at a glance
- **Data-driven**: Base all visualizations on real research
- **Accessibility**: Ensure good color contrast, readable text
- **Mobile-friendly**: Graphics should work on small screens

### Tools Quick Start:
- **Canva**: Search for "infographic templates" to start
- **Heroicons**: Download SVG icons, change color in code
- **Chart.js**: Already implemented - just update data in charts.js
- **Screenshots**: Use browser dev tools to capture Chart.js output

### Free Image Resources:
- Unsplash (photos)
- Pexels (photos)
- Undraw (illustrations)
- Heroicons (icons)
- Flaticon (icons - attribution required for free)
